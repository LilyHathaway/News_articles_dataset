Discovery危险实验 | 重庆大巴坠江，汽车落水如何逃生？​​

2018年10月28日，重庆大巴坠江事故引发全民关注。当时很多网友都在谴责小轿车女司机，认为一定是她逆向行驶，才导致了这次事故。
但现实却反转了，大巴后方车辆的行车记录仪拍到了决定性的一幕：
是公交车在行驶中突然越过中心实线，撞击对向正常行驶的小轿车后冲上路沿，撞断护栏，坠入江中。



事故真相警察还在查询，而小编今天主要想说的是：汽车落水自救。
意外来的非常突然，想必大巴司机和车上乘客也根本想不到“坠江”会发生在自己身上。
我就曾经亲眼看过有游客在景区湖边擦车，可能是手刹没拉或者泥沙太软吧，车缓缓滑进了水里......
所以大家也千万不要抱着侥幸心理，技多不压身，求救不如自救，而且车辆落水之后，给你的时间可能就短短的30-12秒而已。



由于公交大巴普遍配有救生锤，所以我们今天主要讲轿车落水逃生。
国外节目《流言终结者》做过实验，结果表明，在车刚刚入水的时候，开车门是非常容易的，这时候能跑赶紧跑，命都要没了，就别心疼车了。
<<  水刚淹过鞋子，门轻松打开了  >>
<<  水刚淹过膝盖，开门已经很费力  >>

<<  水漫过胸口，门无法开启  >>
由于重量的分布，车辆入水一般都是车头先向下沉。
假如车门宽1米，水漫过车门到车窗位置，这时推动车门就相当于举起250公斤物体重量，被困在车内的人是不可能把车门推开的。如果水漫过车门一半，不到车把手的位置，要打开车门就相当于举起50公斤的物体重量，这个重量对女性来说还是有些困难的。
等到车身完全没入水后，车门两侧压力基本持平，车门又能被顺利打开。
不过很多人一落水就会拼命挣扎，无法保持冷静预留足够的氧气，在撑到车完全沉入水里之前就会窒息而死。（实验人员第一次入水时，也是十分惊恐，用尽全力想打开车门，结果没能等到车内外压力相等，肺里就没有氧气了。）
有经验的幸存者会知道，一旦落入水中，不管是手动车窗还是电动车窗，统统都失灵了，这是为什么？
实验人员采用水深两尺时，车窗会受到的压力，350磅，开始了这项实验：
<<  350磅的压力，手动车窗纹丝不动  >>

<<  350磅的压力，电动车窗挣扎了一小下  >>
实验表明，手动车窗，就算摇杆都断了玻璃也不会动一下；而电动车窗在水下是可以正常工作的，短时间内不会断电短路，但是在水中，车外会有极大的压力，把玻璃向车身方向推，死死得按住玻璃，就好像大象踩住了蚊子一样吧...
所以想通过开窗的方式出去，你可能只有几秒的时间，在水漫过窗沿之前打开它。
开窗很困难，那只有选择破窗而出了。
这比你想象的容易很多很多，前提是你有合适的工具。
实验人员先用我们在车内最容易获得的车钥匙和手机做了尝试，结果当然失败了，连一丝划痕都没留下，不知道诺基亚手机能不能行。
<<  成年男子穿着钢头皮鞋站在车窗上  >>
钢头皮鞋一般人其实不会穿的吧...这样一双鞋对着车窗又踢又踹，再加上一个成年男子的体重，车窗依旧坚挺...
<<  公交大巴和火车上常见的救生锤  >>

<<  弹簧打孔机  >>

<<  加油向未来 节目截图  >>
除了《流言终结者》介绍的工具，央视综艺《加油向未来》也有过一期节目专门讲解“汽车进水如何逃生”。
<<  踩都踩不烂的车窗，轻轻一击就碎了 >>
之所以这么容易，是因为车窗是强化玻璃，经过特殊热处理，因此玻璃表面受到压缩，一旦弄破表面，张力就会消失，这个原理其实和徒手捏鸡蛋类似。
这些工具都非常的小巧，而且操作很简单，x宝50元以下就能买到，在车里放一个，有备无患！
网上流传着这样一个说法：后备车厢有一个应急开关，轻轻一按就能逃出车内。
是真是假呢？
市面上绝大多数轿车都是前置发动机，车子落水后，较重的车头下沉得会比较快，车尾往往会翘起来，这就为驾驶员从后备箱逃生创造了机会。
在汽车的后备箱盖内侧，你可能看到一个黑色的小盖子，打开这个盖子，可以看到里面有一个白色的拉扣或开关。用力拉动拉扣或拨动开关，后备箱就可以被打开了，这就是我们所说的“后备箱逃生装置”。

看起来似乎非常靠谱，但存在一个问题：我国关于汽车门锁和保持件的国家标准里，并没有强制性地要求这样做。
也就是说：后备箱逃生装置并非“标配”。
不是每个牌子的车都有应急逃生装置，而且后备箱里可能会堆行李，座椅也未必放得下去，可实施性比较低，最好的还是带一个小撞针。

总结一下：

PS：车窗刚被打破时，水会大量迅速涌入车内，为防止被碎玻璃划伤，建议等水压平缓再打破车窗。
PS：有一定的风险，属于没有办法的办法。
