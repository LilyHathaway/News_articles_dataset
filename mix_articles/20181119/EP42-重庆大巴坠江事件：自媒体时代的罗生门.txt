事件回顾
—
“重庆大巴坠江事件全回顾”

2018年10月28日10时08分，重庆市万州区一公交车在万州长江二桥桥面与小轿车发生碰撞后，坠入江中。
2018年10月29日14时20分，经公安机关走访调查并综合接报警情况，初步核实失联人员15人（含公交车驾驶员1人）；10月31日0时50分，黑匣子打捞出水并交给当地公安部门； 10月31日23时28分，重庆万州长江二桥坠江公交车被打捞出水；11月1日15时，已找到13名遇难者遗体，身份已全部确认，仍有2人失联 。
2018年11月2日，公交车坠江原因公布，据车内黑匣子监控视频显示，系乘客与司机激烈争执互殴致车辆失控。
事件第一阶段，由于现场照片及第一手报道的误导，舆论将指责的矛头对准“逆行、穿高跟鞋的‘女司机’”；调查结果澄清后，舆论风向一转，群情指责的对象在“女乘客”、“男司机”、“袖手旁观的乘客”和“煽风点火的媒体”间游走。
在普及了十分钟的交通安全知识后，我们不禁开始思考，
// 媒体的业务是否应该在确认第一时间报道信息的时效基础上确保信息的尽可能正确？
// 在自媒体高度发达的今天，传统媒体的作用是否正在逐步被消解？
// 更多曾经并未掌握过话语权的“自媒体大V”也是否需要更多笃定博出位的言论来吸引眼球？
// 当抢头条成为一场赌博，媒体们开始统一口径，为了追求流量而放弃信息真实性的追求时，媒体的公信力又该如何呈现？
// 网民们在一场又一场的真真假假的新闻罗生门中扮演着怎样的角色？
// 网民评论构建的舆论生态是如何影响着媒体的报道风向？

改变中国的一场“罗生门”
—
“从南京彭宇案说开去”

指2006年11月20日早晨，彭宇扶了一位跌倒的老人，在诉讼过程中，老人指认撞人者是刚下车的小伙彭宇，彭宇则予以否认。
在往后几年的逐步调查中，又爆出彭宇确为撞人者的证据，然而事件真相已不可考。但无论事件的真相究竟如何，法官判决时的那一句“不是你撞的人你为什么要扶”的判断已经成为了改变中国生态的转折点。
从彭宇案始，中国民众再也不敢“搀扶老太太”。
在媒介语境中，人性的贪与恶被无限放大与发酵。
 
于是今天的案件中我们可以看到不少复仇心理分析与阴谋论。联想到小众电影《车四十四》、《羊脂球》和揭露人性之恶的韩国电影，真正的人性在善恶光谱中处于什么样的位置呢？

车四十四
或许是笃信了人性之恶，再看回今天的种种，我们仿佛可以发现网络舆论的“戾气”在日益膨胀。媒体与社交网络是否打开了暴戾言论的“潘多拉之盒”？

群体的旁观与失语
—
“危机自救法则”

在大家心照不宣地意识到群体在危险事件中的旁观与失言行为后，我们为大家整理交代了几条保命信条：
1.     少看热闹（来自主播珂珂）；
2.     一旦遇到危险，将看热闹的旁观者involve进来（来自《延禧攻略》、《白夜追凶》与“北京和颐酒店事件”）。
虽然以上两条有些冲突，但不管怎么样相信就对了。
​
传统媒体功能的弱化与新媒体的兴起
—
“论调查报导的死掉”

自媒体时代不难发现，深度长篇的调查报道已经逐步凋亡。一个严谨的、追求真相的、无所畏惧的新闻时代已经过去，取而代之的是一个同质的、碎片化的、追求新鲜感与刺激感的阅读时代。
市场对于新闻的口味正在改变，媒体业的生态也在逐步改变。过去优秀的调查记者纷纷跳槽，新闻理想再也无法使之果腹。新进的记者们或许怀揣着理想，但没有深度的观察与调查指引，中国传统媒体的出路会在哪里？
而自媒体的话语V们，如百度开通的“百家号”，头条的“头条号”，微信的“公众号”，操作逻辑无非是“母爱逻辑”，在大数据背景下算法推送着一条又一条令用户产生爽感的钟爱话题。其后果，只能是“制造区隔”、“加深偏见”，大众媒介的传播范围越广、算法推广的程度越高，是否越代表“人类在技术的安乐窝里，思考能力被质子锁死”？或许人工智能的“觉醒”、“脑机互联”的出现真的会成为可能。

公众心理探究
—
“公众的心态和诉求”

// 健忘
疫苗事件、幼儿园虐童事件、陶崇园事件，公众在一次又一次的爆炸性新闻中日益健忘。当代危机公关仿佛变得容易起来，只需要不回应+删帖删号+默默等待，新的新闻总会层出不穷，总会覆盖掉现有的热点。
 // 猎奇
自媒体时代的发声没有门槛。曝光率、转发量、存在感，成为互联网时代的个人造谣传谣的心理诱导。
一个健康的媒介生态，需要媒体与公众的共同努力。至于媒体的出路在哪里，大家不妨最后喝碗鸡汤，自行搜索《时间的朋友》。
